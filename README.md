A console-based Password Manager application built using C++, which allows users to securely store, manage, and retrieve account credentials. The application supports
user registration, login, password strength checking, and persistent storage using file handling.

 Features :

   -User Registration & Login
   
   -Password Strength Checker (Weak / Medium / Strong)
   -Email Validation for Gmail/Email accounts
   -Add new account credentials
   -Update existing passwords
   -Delete saved credentials
   -Search for stored users
   -Persistent data storage using text files
   -Display all stored accounts
   -Secure logout with auto-save
